# Конвертер валют (Currency Converter)

Консольное приложение для конвертации валют с использованием актуальных курсов из API.

## Возможности

- **Конвертация валют** — ввод суммы, выбор исходной и целевой валюты
- **Просмотр доступных валют** — список всех поддерживаемых валют
- **Обновление курсов** — загрузка актуальных курсов с open.er-api.com
- **Локальное хранение** — курсы сохраняются в `currency_rates.json` и доступны без сети

## Требования

- Python 3.10+
- requests

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python currency.py
```

или

```bash
python cli.py
```

## Структура проекта

```
├── api_client.py      # API-клиент (get_currency_rate, update_currency_rates)
├── storage.py         # Хранилище (load_rates, save_to_file)
├── cli.py             # Консольный интерфейс (main, меню, convert_currency)
├── currency.py        # Точка входа
├── currency_rates.json # Кеш курсов валют
├── requirements.txt   # Зависимости
└── README.md
```

## Использование

1. При первом запуске выберите пункт **3** для загрузки курсов валют (требуется интернет).
2. Для конвертации выберите пункт **1**, введите сумму и коды валют (например, USD → EUR).
3. Пункт **2** показывает все доступные валюты из `currency_rates.json`.
